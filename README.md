# ERP_Delphi
Este é um projeto pequeno de um ERP feito em Delphi com Banco de Dados Firebird. 

O projeto conta com 5 forms principais de acesso ao Usuário.

*Tela de Login

*Tela Principal
  ->Menu
    ->Cadastro de Pessoa
    ->Cadastro de Produto
    ->Cadastro de Pedido
    -> Importar Arquivos .csv, .xml e .txt

Fora os forms Principais, o projeto conta com units que visam a Metodologia Ágil, Organização do projeto e entendimento claro dos objetivos, que são elas:

*udmDados - Unit onde se encontra as conexões com o banco de dados e as querys principais utilizadas no projeto.
*uLogSistema - Unit responsavel por criar registro log no banco de dados das ações realizadas pelo usuário durante o uso do sistema.

Para desenvolver este projeto optei por escolher Delphi por ser uma tecnologia qual eu possuo domínio e tenho muita experiência e conhecimento, e o FireBird alinhado ao IBExpert que hoje é uma das melhores opções para projetos e empresas.

No delphi, foi utilizado a própria VCL disponibilizada pela ferramenta.

Nos arquivos deste repositório, está anexado todos os códigos fontes criados no projeto, imagens para logo, exemplos para testar importação dos dados e a Base de dados em si utilizada durante o desenvolvimento do projeto.

Por fim, com o desenvolvimento desse projeto, pude colocar em pratica ainda mais os meus conhecimentos, visto que foi necessário desenvolver do zero e configurar todo o ambiente de trabalho para chegar a este resultado.
